# webpack-start

### Це стартова конфігурація Webpack

Що він вміє:
- генерує `index.html`;
- підключений `modern-normalize`;
- працює з CSS/SASS (додає файл стилів у `index.html`);
- є автопрефіксер;
- підключений `babel`;
- додає локальні шрифти в `build`, папка `fonts`;
- додає картинки в `build`, папка `img`;
- "зжимає" картинки.
